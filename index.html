Magento

Massive e-commerce websites

Magento, Inc

Magento is an open-source e-commerce platform written in PHP. It is one of the most popular open e-commerce systems in the network. This software is created using the Zend Framework. Wikipedia

Angular

Web-Sites & Web-Applications

Google

Angular is a TypeScript-based open-source web application framework led by the Angular Team at Google and by a community of individuals and corporations. Angular is a complete rewrite from the same team that built AngularJS. Wikipedia

WordPress

Blogs & e-commerce websites

WordPress Foundation

WordPress is a content management system based on PHP and MySQL that is usually used with the MySQL or MariaDB database servers but can also use the SQLite database engine. Features include a plugin architecture and a template system, referred to inside WordPress as Themes. Wikipedia

ASP & MVC

Web-Sites & Web-Applications

Microsoft

The ASP.NET MVC is a web application framework developed by Microsoft, which implements the model–view–controller pattern. It is open-source software, apart from the ASP.NET Web Forms component which is proprietary. Wikipedia
